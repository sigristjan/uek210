Cloud

Ich weiß, dass es unterschiedliche Arten von Clouds gibt:

  • Private Cloud
  • Public Cloud
  • Hybrid Cloud

Die verschiedenen Cloud-Arten haben alle unterschiedliche Zwecke und Anwendungsfälle.

Cloudlösungen

  • IaaS: Infrastructure as a Service
  • PaaS: Platform as a Service
  • SaaS: Software as a Service
  • FaaS: Function as a Service

Die unterschiedlichen Cloudlösungen beschreiben, was alles der Cloudanbieter an Service anbietet.

Cloud merkmale

  • On-Demand self-service

    Steht sofort zur Verfügung und man kann die benutzten Sachen einfach bearbeiten

  • Broad network access

    Man hat von überall auf den Service Zugriff. Verbergt eine angeschlossene Cloud/Pool/Clustor. Es ist dadurch nicht ersichtlich wie viele Nodes es hat. Beim Einganspunkt des Clusters/Cloud gibt es einen LoadBalancer.

  • Ressource pooling

    Rechner werden geteilt → lohnt sich, wenn viele Server zur Verfügung sind (Economics of Scale)

  • Rapid elasticity

    Ressourcen können bei hoher Auslastung automatisch erweitert werden.

  • Measured service

    Der effektive Verbrauch wird abgerechnet.

Vorteile

  • Bezahlung nach Verbrauch
  • Geringe Anschaffungskosten
  • Geringer Wartungsaufwand
  • Weltweiter Zugang
  • Garantierte Betriebskontinuität und Verfügbarkeit
  • Hohe Flexibilität und Elastizität
  • Kurze Markteinführungszeit (schneller App-Launch)

Nachteile

  • Bezahlung nach Verbrauch (hohe Kosten)
  • Sehr hohe Komplexität
  • Weniger Kontrolle
  • Datenhoheit
  • Gebunden an den Anbieter
